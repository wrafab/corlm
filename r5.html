Project Description:


What are the design rules and limitations for the MakerBot Replicator 5th Gen printers?
Design Rules: -You must use certain applications like Fusion360 to design the object you will be printing. -After designing, you must export the file to MakerBot in order to print it.
Limitations: -Only print to the size of the 3D printer. -Filament runs out over time. -The base of the object has to be the size of the entire object. -Objects such as trees must have supports all the way up when printed. -The printer cannot capture small, intricate details.
